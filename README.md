# KPMG Data Analysis using Excel

## Project Overview
This project focuses on customer demographics, transaction analysis, customer segmentation, and Customer Lifetime Value (CLV) analysis using Microsoft Excel.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Power Query
- Dashboard Creation
- Data Visualization

---

## Project Tasks

### Task 1 — Data Cleaning
- Removed duplicate records
- Standardized gender and state values
- Handled missing values

### Task 2 — Customer Segmentation
- Wealth segment analysis
- Gender analysis
- Industry segmentation

### Task 3 — Transaction Analysis
- Monthly sales trends
- Product performance analysis
- Top customer identification

### Task 4 — New Customer Insights
- State-wise customer distribution
- Revenue opportunity analysis

### Task 5 — Customer Lifetime Value (CLV)
- Calculated CLV using Excel formulas
- Compared CLV across customer groups

### Task 6 — Dashboard & Recommendations
- Interactive dashboard creation
- KPI analysis
- Business recommendations

---

## Dashboard Features
- KPI Cards
- Interactive Charts
- Dynamic Slicers
- Customer Insights
